# 直线的方程

<myexercise>
    <p>过定点 $P(1,2)$ 的直线 $l$ 在 $x$ 正半轴与 $y$ 正半轴上的截距分别为 $a,b$, 求 $ab$ 的最小值.
</p>
</myexercise>
<mysolution>
    <p>    由已知, $a,b>0$ 且可设 $l\colon \dfrac{x}a+ \dfrac{y}b= 1$, 则
    \[\frac1a+ \frac2b= 1\geqslant 2\sqrt{\frac2{ab}},\quad
    ab\geqslant 8.\]
</p>
</mysolution>
